# SQL Data Warehouse Project

## Overview

This project demonstrates the designing and implementation of a SQL Data Warehouse for retail sales analytics.
This project follows a layered architecture (Bronze, Silver, Gold) to extract, transform, and load raw business data into analytics-ready tables.
The final data model supports Exploratory Data Analysis(EDA), customer analytics, reporting, product performance analytics and business use cases.

## Objectives

- To build a SQL-based Data Warehouse
- To implement ETL (Extract, Transform, Load) pipelines
- Practicing real world data engineering project

## Data Warehouse Architecture

Raw Data(Source CSV files)
    ↓
Bronze Layer
    ↓
Silver Layer
    ↓
Gold Layer
    ↓
Analytics & Reporting
